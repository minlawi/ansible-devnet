### Create User for SSH Key-based Method

**Default user (ansible_admin):**
```bash
ansible-playbook 03_ssh_key_auth/configure_ssh_key_auth.yml
```

**Custom user (e.g., lawi, network_admin, etc.):**
```bash
ansible-playbook 03_ssh_key_auth/configure_ssh_key_auth.yml -e "target_user=lawi"
ansible-playbook 03_ssh_key_auth/configure_ssh_key_auth.yml -e "target_user=network_admin"
```

### Check if User Exists in Managed Nodes

**Check default user:**
```bash
ansible-playbook 03_ssh_key_auth/check_ssh_key.yml
```

**Check custom user:**
```bash
ansible-playbook 03_ssh_key_auth/check_ssh_key.yml -e "target_user=lawi"
```

### Remove User and SSH Pubkey-chain

**Remove default user:**
```bash
ansible-playbook 03_ssh_key_auth/remove_ssh_key_auth.yml
```

**Remove custom user:**
```bash
ansible-playbook 03_ssh_key_auth/remove_ssh_key_auth.yml -e "target_user=lawi"
```

### Test SSH Key Authentication

**Connect using your configured user:**
```bash
ssh -i ~/.ssh/cisco_rsa ansible_admin@<device_ip_address>
ssh -i ~/.ssh/cisco_rsa lawi@<device_ip_address>
```
