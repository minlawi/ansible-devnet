### Ansible command lists
---

`1. View ansible configuration`
```bash
ansible-config -h 
```

`2. REPL console for executing Ansible tasks.`
```bash
ansible-console -h 
```

`3. Perform various Role and Collection related operations`
```bash
ansible-galaxy -h
```

`4. Runs Ansible Playbooks, executing the defined tasks on the targeted hosts.`
```bash
ansible-playbook -h
```

`5. Plugin documentation tool`
```bash
ansible-doc -l
```

`6. Ansible inventory information, by default it uses the inventory script JSON format`
```bash
ansible-inventory -h
```

`7. Pulls playbooks from a VCS repo and executes them from the local host`
```bash
ansible-pull -h
```

`8. Encrypt/Decrypt utility for Ansible data files`
```bash
ansible-vault -h
```
